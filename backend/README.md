## Inventory Control - Backend

---

#### To run individually (without docker):

- you need to have Nodejs installed;

- create in the root of project the .env file (follow the .env.example);

- install the depencies:

  ```bash
  npm i
  ```

- run:

  ```bash
  npm run start:dev
  ```

#### To run the tests

- without coverage:

  ```bash
  npm run test
  ```

- without coverage:

  ```bash
  npm run test:cov
  ```
